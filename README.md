# 🤖 BOT-ZAP: Gestão & Agendamento Clínico

O **BOT-ZAP** é uma solução de automação para WhatsApp desenvolvida para otimizar o atendimento inicial e o agendamento de consultas em clínicas e consultórios. O bot atua desde a triagem inteligente até a confirmação de horários, reduzindo a carga de trabalho da recepção.

---

## 🚀 Funcionalidades Principais (Planned/Released)

* **📅 Agendamento Inteligente:** Consulta de horários disponíveis em tempo real.
* **🩺 Triagem Inicial:** Direcionamento do paciente com base na especialidade desejada.
* **🔔 Lembretes Automáticos:** Redução de faltas através de notificações pré-consulta.
* **📂 Integração de Prontuário:** (Se houver) Sincronização básica de dados do paciente.
* **💳 Informações de Pagamento:** Envio de chaves PIX ou convênios aceitos.
